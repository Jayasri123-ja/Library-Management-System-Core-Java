## Description
This is a **console-based Library Management System** implemented in **Core Java**.
It allows users to **add, view, issue, and return books** using a **menu-driven interface**.
The project demonstrates **object-oriented programming (OOP)**, **collections**, and **basic Java concepts** without any frontend or web frameworks.

## Features
* Add new books with **ID** and **name**
* View all books with their **status** (issued or available)
* Issue a book to a user
* Return a book
* Menu-driven program for easy usage

## Technology Used

* **Java (Core Java)**
* **ArrayList** for storing book data
* **Scanner** for user input

## How to Run

1. Download or clone the repository:
   git clone <your-repository-link>
  
2. Open **Command Prompt** and navigate to the project folder:
   cd Desktop\LibraryManagementSystem
 
3. Compile the Java files:
   javac library\Book.java library\Library.java
  
4. Run the program:
   java library.Library
  
5. Follow the **menu instructions** to add, view, issue, or return books.
